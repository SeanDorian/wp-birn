SubZane Member Pages is a free membership management system for WordPress&reg; that restricts content to registered users.

== Description ==

SZ Member Pages is a free plugin that can set restrictions on specific pages to certain User Roles or users. On every page you can define what user roles or users that are allowed to view a specific page. The security settings can be inherited by the page children. Use this together with <a href="http://wordpress.org/extend/plugins/user-role-editor/">User Role Editor</a> to create new User Roles for your website. 

= Features: =

* Can restrict visitor access to specific pages and its child pages to certain User Roles or users
* Redirects restricted pages to default WordPress Login Screen
* Works well with WordPress Network (Multi)

== Installation ==

SubZane Member Pages is designed to run "out-of-the-box" with no modifications to your WP installation necessary. Please follow the installation instructions below. 

1. Upload the `/SZMemberPages/` directory and its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress&reg;

You are ready to begin using SubZane Member Pages!
